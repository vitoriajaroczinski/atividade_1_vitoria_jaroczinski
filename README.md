# atividade_1_vitoria_jaroczinski
O objetivo desta atividade é aprender sobre as principais tags de HTML e os seus atribuitos, aplicando-os na prática para reforçar o conteúdo. Para iniciar, foi feito um relatório entre 20 e 30 tags, e depois utilizamos o conteúdo para criar nosso proprio site.
Para abrir-lo, é necessário localizar o arquivo index na pasta, apertar com o botão direito e selecionar a opção "Abrir com" e escolher o Chrome.
